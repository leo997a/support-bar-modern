# شريط الداعمين للبث المباشر

مشروع مخصص لإنشاء شريط داعمين يمكن استخدامه في البث المباشر مع OBS Studio أو أي برنامج بث آخر.

## الميزات

- تصميم عصري وقابل للتخصيص
- خلفية شفافة للدمج السلس مع البث
- دقة شاشة 1920×1080 متوافقة مع البث المباشر
- تحديث ديناميكي للداعمين الجدد
- دعم اللغة العربية
- ربط كامل بين صفحة الإدارة وشريط الداعمين

## ملفات المشروع

### 1. شريط الداعمين (التصميم العصري)
- الملف: `support-bar-modern.html`
- وصف: شريط عمودي متطور يظهر على الجانب الأيسر من الشاشة بتصميم عصري

### 2. صفحة إدارة الداعمين
- الملف: `supporters-admin.html`
- وصف: لوحة تحكم لإدارة الداعمين وإضافتهم مع معاينة مباشرة

### 3. ملف البيانات
- الملف: `support-data.js`
- وصف: يحتوي على البيانات والوظائف الخاصة بإدارة الداعمين

## كيفية الاستخدام

1. افتح ملف `support-bar-modern.html` في متصفحك أو استخدمه في OBS
2. افتح ملف `supporters-admin.html` لإدارة الداعمين
3. أضف داعمين جدد باستخدام صفحة الإدارة
4. سيتم تحديث شريط الداعمين تلقائيًا مع المعاينة المباشرة

## الاستخدام مع OBS Studio

1. افتح OBS Studio
2. أضف مصدر جديد "متصفح" (Browser)
3. اختر المسار الكامل لملف `support-bar-modern.html`
4. قم بضبط حجم المصدر ليكون 1920×1080
5. تأكد من تفعيل خيار "وضع الخلفية الشفافة" إذا كان متاحًا

## التخصيص

يمكنك تعديل التصاميم عن طريق تعديل ملفات HTML مباشرة:
- لتغيير الألوان: عدل قيم الخصائص في أقسام الـ CSS
- لتغيير الأبعاد: عدل قيم العرض والارتفاع في قسم التنسيقات
- لتغيير الخطوط: عدل خاصية font-family في أقسام الـ CSS

## المتطلبات

- متصفح ويب حديث (Chrome, Firefox, Edge)
- خادم ويب محلي (للاستخدام المحلي) أو استضافة ويب

## الترخيص

هذا المشروع مجاني للاستخدام الشخصي والتجاري مع ذكر المصدر.
